About

Contains the meta data about the index including the number of packets, flows, hosts, services and also the duration (in seconds) of the indexed pcaps. 

 xtractr.about.duration
 xtractr.about.packets

Attributes

hosts  [R]  Returns the #hosts in the index
packets  [R]  Returns the #packets in the index
services  [R]  Returns the #services in the index
version  [R]  Returns the version of the xtractr server
